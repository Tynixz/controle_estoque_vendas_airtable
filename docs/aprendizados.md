# Aprendizados

Durante o desenvolvimento deste projeto foram trabalhados conceitos relacionados a:

- modelagem relacional;
- organização de processos;
- automações no-code;
- integridade de dados;
- experiência do usuário;
- estruturação operacional.

---

## Desafios encontrados

### Duplicidade de produtos

Inicialmente os produtos eram cadastrados livremente.

Isso gerava:

- nomes inconsistentes;
- dificuldade de contagem;
- erros de estoque.

A solução foi separar:

- catálogo de produtos;
- unidades físicas individuais.

---

### Controle de status

Outro desafio foi controlar:

- disponibilidade;
- produtos enviados para vendedores;
- produtos vendidos.

A solução foi utilizar automações e status automáticos.

---

## Melhorias futuras

- controle de acesso por perfil;
- login para vendedores;
- dashboards financeiros;
- migração futura para aplicação própria;
- integração com ferramentas externas.
